## Key Algorithms

### 1. Section Detection
```
Input: Resume text (lines)

For each line:
  If line matches "summary|objective|profile":
    current_section = "summary"
  Else if line matches "skills|technical":
    current_section = "skills"
  Else if line matches "experience|employment":
    current_section = "experience"
  Else if line matches "education|degree":
    current_section = "education"
  Else:
    Add line to current_section

Output: {summary: [...], skills: [...], experience: [...]}
```

### 2. Bullet Point Extraction
```
Input: Section lines

bullets = []
current_bullet = ""

For each line:
  If line starts with bullet marker (â€¢, -, *, etc.):
    If current_bullet not empty:
      bullets.append(current_bullet)
    current_bullet = line
  Else if line not empty:
    current_bullet += " " + line  # Continuation
  Else:
    bullets.append(current_bullet)
    current_bullet = ""

Output: [bullet1, bullet2, bullet3...]
```

### 3. ATS Impact Calculation
```
Input: original_bullet, enhanced_bullet, jd_keywords

# Count keyword improvements
original_kw = count_keywords(original_bullet, jd_keywords)
enhanced_kw = count_keywords(enhanced_bullet, jd_keywords)
kw_impact = (enhanced_kw - original_kw) * 1.5

# Count action verb improvements
original_verbs = count_action_verbs(original_bullet)
enhanced_verbs = count_action_verbs(enhanced_bullet)
verb_impact = (enhanced_verbs - original_verbs) * 1.0

# Count metric improvements
original_metrics = count_numbers(original_bullet)
enhanced_metrics = count_numbers(enhanced_bullet)
metric_impact = (enhanced_metrics - original_metrics) * 0.5

total_impact = kw_impact + verb_impact + metric_impact

Output: +2.5 (or whatever the total is)
```
